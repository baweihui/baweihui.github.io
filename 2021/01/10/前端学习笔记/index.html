前端学习笔记

No tag
Publish Date:   2021-01-10

##内容
我是从今年的三月份开始学习前端的.其实一天没有学很长时间平均下来一天也就是四五个小时的时间吧。我只是个普通的大学生找了一个普通的实习。算不上厉害，只是因为自己自学的时候像个无头苍蝇一样浪费了很多时间，所以我想把自己的经验和大家分享一下，因为在我自学的时候也受到过别人的指引。我很感激他们，我也愿意和他们一样去小小的帮助一下有需要的同学。

还有我只是个普通本科大学生。我以下提到的都是根据我自己的真实经历总结的。要是有些大佬觉得我的办法又笨又慢。请嘴下留情。

首先重要的几点

入门最好是直接看视频，尤其是小白这很重要。不然的话你连一个编译器都用不明白(我有一个朋友刚开始用vscode的时候就不会运行html代码)。视频的知识深度会比较浅。但是经过视频创作者的总结和加工也会更加通俗易懂。等你真正对这个部分了解了之后再去看权威的书籍来提升自己。

要跟着打代码！！！不要看视频做了一遍，你就懂了你就会了，一定一定要自己动手试试。一看就会一写就废真不是闹的，有时候一个标点符号都会让你苦恼半天的。所以从你学的第一个html标签第一个css属性第一个js语句开始。一定要自己实现一遍。

无论是看视频还是看书都要有记笔记的习惯，不要简单的觉得我看懂了就行了，首先是你记不住，过几天就忘。其次记笔记，把视频中的或者书中的知识点梳理成自己能理解的语言的这个过程就是很好的学习。一些比较复杂的知识点不是第一次遇到就能真正理解的。把你目前所能理解的程度记在笔记上。随着以后你不断的学习可能对这个知识点有新的认知。然后在笔记中不断的完善。逐步加深你对这个知识点的理解。

不要想着我把哪个部分学的十分透彻了再去学下一个部分，相信我无论是你看的视频还是书，看一遍你绝对记不住。记不住回去看自己的笔记，然后第二次遇到你没准还记不住。没关系再回去看自己的笔记。反反复复的才能真正的记住。

一. html和css部分

这两个部分入门就是看视频就行了，随便在b站搜一些入门视频都可以。我当时看的就是b站的一个叫渡一教育的培训班的公开课(在我的一个叫it的收藏夹中有这个课)，说真的这个讲师成哥真的很有才华，你会爱上听他的课。当然只是建议。他的视频会经常聊其他的东西，很有趣反正我爱听你们要是看两集不喜欢的话，可以自己去看看其他的htmlcss入门课程，b站应该有其他家的html和css入门视频吧。我看的这个课在讲完基础的课程之后会有另一个老师带领大家做一个静态的淘宝官网。这老师也挺好的。干货很多。让你知道有些东西原来还可以这么玩。很神奇。相信我看完这个之后让你重新写一遍淘宝官网你还是不会写。没关系的。再去看2017年的百度前端教程（http://ife.baidu.com/college/detail/id/5）。这个也是从头带你学一遍html和css，但是这次是文字形式的喽，可要打起精神认真学了。这个教程会推荐很多文章让你更深的理解知识点。当然推荐的菜鸟教程和w3c的就别看了。其中会推荐叫MDN的网站，这个一定要好好看，很重要。看完这个你又会对html和css有一个新的认识。不要把目光局限在他推荐的这点文章上，比如你了解到一个叫浮动的东西，你觉得他提供的资料不足以让你很得心应手的使用这个属性，你就去搜搜有关的文章(常见的社区有CSDN。掘金。思否，知乎等等)(好像很多人对于知乎很有争议说是吹牛的太多，但是关于前端的学习路线和书籍推荐方面在知乎上真的有很多高质量的回答，建议你们去看看，个人觉得知乎在这种技术方面的讨论做的还是相当成功的，只能说是你所见的世界就是你自身的影子。你总觉得知乎上吹牛逼的太多我只能说是你把知乎当做一个娱乐软件来对待了，平时总在知乎上看什么明星八卦娱乐那当然吹牛逼的一个赛一个)，总会有一些好的文章让你惊叹原来是这样的。学习最忌讳的就是理所应当。

再次提醒一遍记笔记记笔记记笔记！！！

我的github上有个文件夹就是我这几个月的学习笔记。几十万字应该是有的。

然后试着去自己切图，最好是切那些有讲解的图。就比如前面的那个视频看了有些日子了，忘得差不多了吧。正好别重新看视频了，直接自己上手写看看能不能写出来，要是写不出来再去翻视频找找解决的办法。因为咱们是自学所以不要贸然尝试过于超出自己实力的东西，不然有了问题没人解答很难受的。这个阶段无论你是知道flex还是不知道flex都不要用它，不是它不好用，是它太好用了，你不被普通的css恶心恶心，你是体会不到flex的妙处的。况且有时候不兼容flex的话，你还是学着脱离flex开发。如果你是大佬现在就能独立的切完静态的淘宝首页了，

恭喜你，有关html和css你可以暂时告一个段落了。

这个阶段真的不适合看什么书，没必要。尤其是上来就看什么html5权威指南，好几百页，你没看完你就转行了。

html的话没什么难理解的。其中就是各种表单的属性不太好记而已，还有块级元素和行级元素的定义要理解。

css的话盒模型，浮动，定位，包含块，BFC，margin合并。margin坍塌，选择器的权重之类的比较重要。

二 css3

学习完html+css之后，我推荐你开始学一下css3，css3中主要新增了动画，3D,复杂的伪类选择器，背景。文字边框等等，等你学完这个部分之后(我当初看的那个视频没了，你们就在b站搜搜吧，我听说黑马的这个入门视频都挺不错。播放量也挺高。推荐看入门视频的话找一些培训班的课程，毕竟他们就是干这个的，作为入门视频来说无论是难度还是易理解程度都和很不错)，就要开始学习我上面提到的flex。使用flex搭配css3的属性，你就能搭建一个更加美观，更加智能的响应式布局(什么是响应式自己去查)。总之学完这个部分你的任务就是学会做一个响应式布局的网站，当你能做完这个你就可以去学习js了。

三 javascript

javascript的话我看的还是渡一教育的视频（我收藏夹里面有），同样你不喜欢闲话多的可以在b站找一些其他培训机构的js入门视频，反正是入门差不太多的。认真看一定要自己实现一遍老师的代码，并且一定要认真记笔记，文字比较直观，到时候你忘记哪个部分回来看的话笔记也会比视频更直观。这个视频的话我看了大概是三遍。因为这个视频的话讲的深度还是挺深的，我不知道其他的入门视频都讲了什么，反正这个视频的话讲的js的核心还挺多的，其中js的预编译，类型转换，对象的原型，继承，作用域，作用域链，闭包，this的指向问题这些知识点不是看了一遍视频就能理解的很好的，需要不断的通过项目和书籍深化理解。也可能是我比较笨。这个就根据你们自己的实际情况来定，反正我是推荐看完一遍之后再紧接着看一遍，也算是差缺补漏吧。这个视频看完两遍之后你基本上对js有个大概的了解了。个人推荐不要急着学Vue。因为你现在还没有了解ES6学习Vue之前是要了解一部分ES6的。并且要是你的原生就是学的很透彻的话，学习Vue也会非常简单，到时候学Vue的时候学习的是Vue本身而不需要掺杂着大量原生js的内容。学习要循序渐进。js的深入学习的话推荐看书，我当时看的是DOM编程艺术和javascript高级程序设计这两本书。每本书看完之后都会对javasctipt有一个新的层次的认知。DOM编程艺术主要讲的就是有关如何操作DOM来实现编程。比较简单也很重要推荐阅读一遍并且在视频的笔记的从基础上完善你的笔记。最重要的就是javascript高级程序的研读，我当时看的是第三个版本，最近出来四了，可以买来看看。(书籍推荐在当当网上购买，书的质量不错并且经常会有满100减50的优惠活动)，第三个版本的话有很多章节是过时的比如ajax，xml等内容，第四个版本还有没有我不知道，都学到这了你应该清楚什么是重要的什么是过时的就不需要我在一一告诉你了。这本书一定要认真的研读。看完这本书之后你对js的原型，原型链，作用域，作用域链，闭包，this的指向问题应该就会很透彻了(继续在你你视频笔记的基础上完善你的笔记，这时候你会真的发现看视频时你以为你懂了，其实没有真的懂。但是你现在也不会说就完全明白了js这门语言，只不过你现在的水平足以胜任一个前端开发实习生的水平了，对于js这门语言的理解，还需要你在以后的工作生涯中慢慢体会。这也就是我在最前面提到的，不要想着学精了一个部分在接着开始下一个部分。学无止境)。IT行业学习之路漫漫，看完这两本书绝对不是你学习js的终点，但是暂时先告一段落。什么js的设计模式，性能优化，深入理解都先放一放你现在的知识层面还不足以真正理解这些书的奥义。(况且就算你理解了你目前也用不到过不了几天就会忘掉的，你要学习那些你经常能练习到的知识点，学完之后通过练习来记忆来深入理解。过早的学习用不到的难一点的技术。你用不到也记不住)，还是那句话学习需要循序渐进。你现在的重要任务是学会使用jshtmlcss来搭建网页，至于项目的可维护性，优化，设计模式都是在你已经学会基础的前端技术的前提下的进阶内容。

四 html5 ES6

为什么我把HTML5放到这里呢。因为H5里面的很多东西是要配合着js来使用的。等你学过JS之后再学H5会比较好。H5主要有视频，音频，canvas，svg，本地存储，拖放，这部分也是看入门视频学习就行了。等你学到了canvas和svg你就会知道其实这两个在初级前端开发阶段用到的很少。除非你是确定了要做H5方向的前端。不然的话了解一下这两个是干嘛的就行。以后真的有需求了在学。

ES6很重要在学习Vue之前一定要提前了解一下ES6的部分语法。ES6推荐看阮一峰老师的ES6基础入门。写的很不错，但是不推荐从头到尾全看一遍，推荐只看一些使用频繁的部分。let，const，解构赋值，箭头函数，模板字符串，promise，module。看不懂没关系我下面会推荐看视频来深化他们。ES6中像class，Generator,proxy,symbol等知识点你在哪里真的用到了他们之后再去学习吧。

五 总结回顾

你现在html + css + js + html5 + css3 + 部分Es6语法学过了，其实就可以学习Vue了，但是个人推荐一个课程。腾讯课堂的米修在线机构有一门课叫20个原生js实例的课程（付费课程几十块钱吧），这个课程就是完全脱离框架，全部使用原生的js（ES6） html （5）css（3）来搭建网页。这个视频的优点是使用到的技术很全，缺点就是他不会告诉你为什么使用这个，不会去给你一一讲解每条语句的含义。我当时买完挺后悔的因为跟不太上。但是我当时看这个视频的时候只是看了html + css + js的入门视频。都没有接触到ES6和，css3，html5.也没有系统的看过js的书籍。所以说学起来很吃力，但是当我后来开始学ES6，css3，html5的时候才发现，原来好多东西在那个视频中都涉及到了，只不过是我当时没学过而已。所以说这个视频当做学习视频是不可以的。但是如果你已经学过了html + css + js + html5 + css3 + 部分Es6语法的话，再看这个视频可以达到查缺补漏的作用。这样就能保证在你学Vue的时候，学习的更多是Vue本身而不是掺杂着很多原生的js。而且学到这个阶段最起码也得四五个月了吧。你除了静态页面的话，应该也没写过很多涉及到js的页面。难道你就不想看看你现在掌握的知识到底能做一些什么了吗？反正我当时是十分好奇。只不过我好奇的太早了，这个视频和我当时的实力不匹配，导致学习起来很吃力，效果不是很好。这也是我在最上面写到的，不要学超出你实力很大的课程，你消化不了。循序渐进一点一点的增加难度才是进步最快的。

而且这个视频最重要的还是，让我把我的知识转变成了看见的项目，能给人很大的自信和满足感。不然的话学了好几个月。也不知道自己能做点什么。真的很受挫。

六 Vue及其相关

为什么我之前一直在强调要把基础打好再来看Vue。因为使用Vue的话你一定要涉及到node，webpack，npm，Vue cli，模块化，mvvm，git，github这些你之前都没听说过的概念，人面对未知都是恐惧的。这些烦人的概念已经让人很头疼了。所以说先把基础打好在学Vue。不要自己给自己设置难度。而且你只有用原生的js写过代码，才知道Vue的神奇和方便之处。由衷的赞叹尤大大真的是太牛逼了(尤大大是Vue的作者)。这就像flex一样，你没有被浮动布局恶心过，你不会了解flex的便利。

尽管大家都说Vue的中文文档写的很好，我也承认Vue的文档写的很不错。但是我还是觉得不太适合小白。我当时学Vue就是直接看的Vue的文档，因为他的文档里面涉及到很多专业的名词和表达方式，说实话我是真的不太能全部理解文档的意思。我当时前后看过两遍都效果不是很好。

有关Vue力推b站的王红元老师的课程（同样我的b站收藏夹里面有）html和css的课程你可以看其他的，但是这个Vue你一定要看王红元老师的。codewhy永远的神。我是看过Vue的文档，然后也自己折腾过一段Vue之后才了解到的王红元老师的课程。在此之前我花了300多买过一个Vue的课程，看完之后似懂非懂吧。还是畏手畏脚不敢自己去做项目。因为没有太了解Vue。但是看了红元老师的Vue课程，那叫一个相见恨晚呀。我要是早点看见这个视频就不会走那么多弯路了。老师讲的真的是太通透了。真的是一层一层剥茧抽丝。而且这个视频看的小伙伴很多，你遇到很多问题其实早就有人遇到过并且在弹幕中解答了。十分的银杏。看完这个视频我对Vue有了全新的认识。

这个视频分教程部分和实战部分。推荐看完教程部分之后就可以去看Vue的官方文档了。因为你现在有了一点Vue的基础。并且是看的codewhy的视频，到时候看Vue的官方文档还不是手到擒来。

有一说一尽管老师讲的特别好，但是那个项目的话，有一点点瑕疵，就是使用了一个better-scroll的插件，那个插件问题太多了，有很长时间都在处理这个插件带来的问题。不过在处理这个插件的过程中你会学到很过的东西。只不过这个项目不适合当做简历上的项目经历写进去。但是看还是一定要看的。

七 在学一个Vue的课程

个人推荐还是在看一个Vue的视频比较好。每个老师都会有自己的风格，你可以看免费的。也可以去慕课网或者腾讯课堂之类的平台找一些高质量的付费课程。我当时看的也是上面提到的一个叫米修在线的Vue视频。是从零搭建一个饿了么的移动端app。这个课的优点还是涉及到的知识很全，干货很多，老师有认真备课，不会出现逻辑混乱的现象，缺点就是不会为你一句一句的讲解代码的作用，不适合当做入门视频，但是等你有了基础，想要加深锻炼一下自己还是很不错的。这个不是我黑米修在线。我提到的这两个课确实不适合小白，我当初在这两个课上面浪费了很多的时间，这并不是米修在线这个机构的问题，而是我当时实力不够，知识储备不够不能理解消化老师的内容，但是你在此之前已经有了Vue的基础了，想要提升一下自己。那么这个项目是完全没有问题的。这也是我自学的时候浪费时间的重要原因，不知道自己的实力匹配哪种深度的学习视频，这个就需要大家自己去掌握了。

八 你自己的舞台

学到了这你也就真正的入门前端了，你应该学会有自己的想法和安排了。如果着急找工作的话你就要开始背面试题了。推荐牛客网，上面全是面试题目的分享。如果你要是现在找工作不方便的话。你可以学学进阶的知识点，Vue3最近出来了，学一学。node很重要学一学，css觉得很恶心sass，less学一学。要是想要进大厂的话，数据结构和算法要求也是很高的。学一学，每个人学会去找适合自己的方案，毕竟我也才走到这一步，这周二入职，公司的技术栈的Ts + Vue + postcss。我也是正在学。学到这个阶段如果你还有精力的话，就可以看我上面提到的有关js的性能优化，设计模式，可维护性相关的书籍了。现在你需要考虑的不仅仅是如何把网页实现，而是要考虑到如何更加悠亚的实现代码。悠亚永不过时。

以上就是我，一个普通的不能再普通的本科生的前端学习历程，我只是把我的经历分享出来，大家觉得有用的可以借鉴，觉得我的方法又笨又傻的也请别喷我。如果我的这篇文章成为你前端学习的起点，那么我很荣幸。但是记住无论是哪篇文章那个都不会是你前端学习的终点。学习无止境。
